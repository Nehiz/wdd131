Learning Tracks

Abacus Math Track

Best for ages 6-10, this track uses the abacus to enhance mental math skills. Benefits include improved concentration and speed. Ideal for beginners, with hands-on tools and worksheets.

Coding Track

For ages 8-12, kids learn coding through projects like simple games. An example project is a number-guessing game, fostering problem-solving skills with interactive tutorials.

AI Track

Suitable for ages 9-12, this introduces AI with concepts like pattern recognition. Kids explore simple AI games (e.g., Guess the Animal) through interactive activities.
